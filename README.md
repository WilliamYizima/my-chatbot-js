## **Production mode**
#### **Using npm:**

```JSON
npm i
npm run build
npm start
```
#### **Using yarn:**

```JSON
yarn install
yarn build
yarn start
```

### View in browser: http://localhost:8080
<br />
<br />

## **Development mode**
#### **Using npm:**

```JSON
npm i
npm run start:dev
```
#### **Using yarn:**

```JSON
yarn install
yarn start:dev
```

### View in browser: http://localhost:9090
<br />
<br />
